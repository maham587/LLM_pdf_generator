
# 🚀 LLM Document Generator

A simple full-stack application that generates structured documents from a user-defined topic using a local LLM.

---
<img width="1110" height="781" alt="demo" src="https://github.com/user-attachments/assets/18a5f76c-4248-46a8-8324-257097ab1a31" />
## 🧠 Features

* Generate structured content from any topic
* Uses a local LLM (LLaMA via Ollama)
* Clean and minimal React interface
* Fast and pragmatic full-stack implementation

---

## 🛠️ Tech Stack

* **Frontend**: React
* **Backend**: Node.js (Express)
* **LLM**: LLaMA 3 via Ollama
* **API**: REST

---

## ⚙️ Installation

### 1. Clone the repo

```bash
git clone https://github.com/your-username/llm-doc-generator.git
cd llm-doc-generator
```

---

### 2. Start the LLM (Ollama)

```bash
ollama run llama3
```

---

### 3. Backend

```bash
cd backend
npm install
node index.js
```

---

### 4. Frontend

```bash
cd frontend
npm install
npm start
```

---

## 🚀 Usage

1. Enter a topic
2. Click "Generate"
3. Get a structured document instantly
